# Система путей для ботов

Система позволяет создавать маршруты патрулирования для ботов.

## Команды

### Создание и управление путями

```
/pvpbot path create <name>
```
Создать новый путь с указанным именем.

```
/pvpbot path delete <name>
```
Удалить путь.

```
/pvpbot path addpoint <name>
```
Добавить точку в путь на текущей позиции игрока.

```
/pvpbot path removepoint <name> [index]
```
Удалить последнюю точку или точку по индексу.

```
/pvpbot path clear <name>
```
Очистить все точки пути.

```
/pvpbot path loop <name> <true/false>
```
Включить/выключить зацикливание пути (по умолчанию true).

### Управление следованием

```
/pvpbot path start <bot> <path>
```
Запустить бота по указанному пути.

```
/pvpbot path stop <bot>
```
Остановить следование бота по пути.

### Информация

```
/pvpbot path list
```
Показать список всех путей.

```
/pvpbot path info <name>
```
Показать информацию о пути (количество точек, координаты).

## Пример использования

1. Создайте путь:
   ```
   /pvpbot path create patrol1
   ```

2. Добавьте точки маршрута (встаньте в нужное место и выполните):
   ```
   /pvpbot path addpoint patrol1
   ```
   Повторите для каждой точки маршрута.

3. Запустите бота по пути:
   ```
   /pvpbot path start bot1 patrol1
   ```

4. Остановите бота:
   ```
   /pvpbot path stop bot1
   ```

## Особенности

- Бот автоматически переходит к следующей точке при достижении текущей (расстояние < 1.5 блока)
- При зацикливании (loop=true) бот идет туда-обратно по точкам
- При loop=false бот идет по кругу от первой точки к последней и снова к первой
- Навигация использует систему обхода препятствий (прыжки, плавание и т.д.)
- Бот продолжает следовать по пути даже во время боя
